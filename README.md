#inddex.html

Kode HTML yang Anda buat berfungsi untuk menampilkan halaman web profil pribadi sederhana yang terstruktur dan modern.
Dokumen diawali dengan deklarasi <!DOCTYPE html> yang menandakan penggunaan HTML5, kemudian pada bagian <head> terdapat pengaturan karakter (UTF-8),
viewport agar tampilan responsif di berbagai perangkat, serta judul halaman “My Profile”.Disni juga bisa menghubungkan font dari Google Fonts yaitu Playwrite
NZ Basic untuk mempercantik tampilan teks, file CSS eksternal style.css untuk mengatur desain dan tata letak halaman, serta Font Awesome melalui <script> agar 
dapat menggunakan ikon media sosial. Pada bagian <body>, seluruh konten dibungkus dalam sebuah div dengan class container sebagai pembungkus utama. Di dalamnya
terdapat komponen profil (div class profil) yang menampilkan foto menggunakan tag <img> serta teks sapaan “Hi, I'm Ghifra” yang diberi class fontsaya untuk
pengaturan gaya font. Selanjutnya terdapat komponen sosial (div class social) yang berisi judul “My Socials” dan beberapa tautan ke media sosial seperti Instagram,
Facebook, Twitter (X), dan GitHub. Setiap tautan dibungkus dengan tag <a> sehingga dapat diklik, dan di dalamnya terdapat div dengan class sosmed yang menampilkan
ikon Font Awesome beserta nama platform media sosial tersebut. Secara keseluruhan, kode ini bertujuan untuk membuat halaman profil pribadi yang rapi, informatif,
dan mudah diakses, dengan tampilan visual yang diperindah melalui CSS, font khusus, dan ikon.

#style css

Pada bagian awal, selector universal * digunakan untuk menghilangkan margin dan padding bawaan browser sehingga tata letak halaman lebih konsisten.Bagian body
diatur memiliki tinggi satu layar penuh (100vh) dengan latar belakang gambar wallpapers.jpg yang ditampilkan di tengah, menyesuaikan ukuranlayar (cover), dan tidak diulang.
Selain itu, body menggunakan flexbox untuk memposisikan seluruh konten tepat di tengah halaman secara horizontal dan vertikal. Class .container berfungsi sebagai pembungkus utama yang menata seluruh komponen secara vertikal menggunakan flexbox dengan jarak antar elemen (gap) sebesar 40 piksel. Selanjutnya, class .profil digunakan untuk mengatur bagian profil agar foto dan teks tersusun ke bawah, sejajar di tengah, dan memiliki jarak antar elemen. Gambar pada .profil img diatur berukuran 150×150 piksel dan diberi border-radius: 50% sehingga berbentuk lingkaran, sedangkan teks judul profil (.profil h1) diberi warna oranye kekuningan, ukuran font yang cukup besar, serta sedikit padding dan sudut membulat agar terlihat lebih menonjol.Bagian media sosial diatur melalui class .social yang menggunakan flexbox vertikal dengan teks rata tengah dan jarak antar item yang rapi. Judul pada bagian ini (.social h1) diberi warna dan ukuran font yang sedikit lebih kecil dibanding judul profil, sedangkan link (.social a) dibuat tanpa garis bawah dan menggunakan warna hitam agar terlihat bersih. Tombol media sosial diwakili oleh class .sosmed, yang memiliki latar belakang warna krem, ukuran tetap, posisi relatif, serta sudut membulat. Di dalamnya, ikon Font Awesome (.sosmed i) diposisikan secara absolut di sebelah kiri agar ikon dan teks terlihat sejajar. Efek interaksi ditambahkan melalui .sosmed:hover yang mengubah warna latar belakang menjadi pink dan teks menjadi putih dengan transisi halus selama 0,5 detik.Terakhir, class .fontsaya digunakan untuk menerapkan font khusus dari Google Fonts yaitu Playwrite NZ Basic pada teks tertentu, sehingga memberikan kesan unik dan personal pada halaman profil. Secara keseluruhan, CSS ini bertujuan untuk menciptakan tampilan profil yang modern, terpusat, responsif, dan nyaman dilihat.
